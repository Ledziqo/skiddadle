# Resource status rules

- `downloadable_official_pdf`: public PDF/checklist/form. Show Preview + Download + Official Source.
- `official_online_portal`: application must be completed on official portal. Show Open Official Portal only.
- `official_requirements_page`: official web page with requirements. Show Open Official Requirements.
- `needs_verification`: hide publicly until checked manually.

Never create fake official PDFs. If no official PDF exists, show the portal or official requirements page.
