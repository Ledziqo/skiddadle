# Deployment guide for PHP shared hosting

1. Finish the project in `php_starter/public_html`.
2. Run `scripts/download_official_pdfs.sh` locally if you want local PDF preview/download files.
3. Upload the contents of `php_starter/public_html` to hosting `public_html`.
4. Confirm `.htaccess` rewrites work.
5. Test `/`, `/forms.php`, `/country/canada`, `/checklist-generator.php`, `/templates.php`.
6. Make sure `storage/uploads` cannot execute scripts.
7. Replace placeholder contact/payment actions with WhatsApp, Telegram, Chapa/Telebirr/manual payment flow.
